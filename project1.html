Iris Flower Classification



Exploratory data analysis

This project analyzes a dataset on iris flowers, having measurements of sepal length (SL), sepal width (SW), petal length (PL), petal width (PW), and classification. It imports libraries for data manipulation, visualization, and plotting. The dataset is read from an excel file, and basic information such as data types and non-null entries is displayed. The code then creates scatter plots, box plots, and bar plots to compare different classifications of the iris flower to its measurements. These visualizations aid in identifying patterns and trends in the data, providing a comprehensive analysis of the dataset. Overall, this project provides a comprehensive analysis of the iris flower dataset and provides useful visualizations to help understand the different attributes of the iris flower.

Classification Models

This part of the project aims at building a classification model to study the different types of iris flower and classify them by observing the attributes sepal length (SL), sepal width (SW), petal length (PL), petal width (PW). The dataset is split into training and testing sets, and logistic regression is performed on the dataset using the LogisticRegression() function. The performance of the logistic regression model is evaluated using the confusion_matrix(), accuracy_score(), and classification_report() functions.

Further more a web application is buit on python flask with the frontend written html/css and backend in python.The application works to get the classification of the iris flower when the four attributes are given as input.
