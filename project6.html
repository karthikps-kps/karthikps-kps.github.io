Wine Clustering Analysis

Performing Clustering Analysis on Wine Dataset

Unsupervised Learning

The project aimed to perform clustering analysis on the wine dataset to identify patterns and hidden insights. The dataset was imported from the local drive and read into the Jupyter Notebook using Pandas library. The dataset consisted of 178 rows and 13 columns containing information on the chemical composition of wines.

After importing the necessary libraries, descriptive analysis was performed on the dataset to get an overview of the data.

The df.columns function was used to list the names of the columns, and it was found that the dataset contained the following variables :

1. Alcohol

2. Malic Acid

3. Ash

4. Ash Alcanity

5. Magnesium

6. Total Phenols

7. Flavanoids

8. Nonflavanoid Phenols

9. Proanthocyanins

10. Color Intensity

11. Hue

12. OD280

13. Proline

Descriptive statistics were obtained using the df.describe() function.

Clustering analysis was then performed on the dataset using the K-means, Agglomerative Hierarchical Clustering, and Density-Based Spatial Clustering of Applications with Noise (DBSCAN) algorithms. The StandardScaler was used to scale the dataset before clustering analysis. Silhouette score was used to determine the number of clusters for K-means and Agglomerative Hierarchical Clustering algorithms.

Finally, a dendrogram was plotted using the sch.dendrogram() function to visualize the hierarchical clustering of the dataset. The results of the clustering analysis were interpreted to identify the patterns and insights in the data.
