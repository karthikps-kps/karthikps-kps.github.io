Loan Prediction

Loan prediction using logistic regression model



EDA and ML Model

The Loan Prediction project aims to predict whether a loan application is eligible for approval or not based on certain features such as gender, education, income, credit history, loan amount, loan term, and property area.

The project begins with importing necessary libraries such as pandas, numpy, seaborn, and matplotlib. After importing the libraries, we load the train and test datasets using pandas read_csv function. We then create copies of the original train and test datasets to perform operations and modifications without affecting the original data.

After that, we visualize the target variable Loan_Status using a pie chart to understand the distribution of loan approval and rejection. We also visualize categorical values such as Gender, Married, Dependents, Education, Self_Employed, and Property_Area using seaborn countplot.

Finally, we preprocess the data by handling missing values and converting categorical data to numerical data using one-hot encoding. We then split the train dataset into train and validation datasets for model training and evaluation.

Now, various machine learning algorithms such as logistic regression, decision tree, random forest, and XGBoost are used to train the models. The performance of each model was evaluated using metrics such as accuracy, precision, recall, and F1-score. Then the best-performing model based on these metrics is selected and used it to predict the test dataset.

In conclusion, the Loan Prediction project is a classic example of binary classification using supervised learning techniques. The project provides an opportunity to learn data exploration, data preprocessing, feature engineering, model selection, and evaluation techniques
